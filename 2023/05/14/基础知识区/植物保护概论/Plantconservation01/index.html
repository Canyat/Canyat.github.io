一、昆虫的识别

昆虫成虫识别特征

  1. 分节（头胸腹）
  2. 一对触角+口器，通常还有单眼和复眼
  3. 胸部是运动中心（三对足+一般有两对翅）
  4. 腹部是生殖代谢中心（生殖器+大部分内脏）
  5. 变态：昆虫在生长发育过程中要经过一系列内部及外部形态上的变化，才能转变为成虫，这种体态上的改变称为变态
    ![](https://cdn.jsdelivr.net/gh/Canyat/blogima@main/1684147462647chong.png)

昆虫的头部

头部有三种形式：

  1. 下口式（蝗虫、蟋蟀、鳞翅目幼虫）
  2. 前口式（虎甲、步甲、草蛉）
  3. 后口式（蝽蟓、蚜虫、叶蝉）
    ![](https://cdn.jsdelivr.net/gh/Canyat/blogima@main/1684147431730tou.png)

昆虫的触角

构造

含柄节 梗节 鞭节

类型

类型多种多样，其变化主要在鞭节处。（常见的有刚毛状、念珠状、羽状、棒状、锯齿状……）

功能

触觉、嗅觉、听觉……

昆虫的口器

1.咀嚼式口器（固体食物）
2.吸收式口器（液体食物）
(1)刺吸式口器
(2)虹吸式口器
(3)锉吸式口器
(4)舐吸式口器
嚼吸式口器（固体+液体）

二、昆虫生物学习性

生殖方式

1.两性生殖 (有性生殖)
a.根据雌虫产出后代的虫态，分卵生、胎生
b.根据雌虫产出后代的个体数，分单胚生殖（一卵一个体）、多胚生殖（一卵多个体）
绝大多数昆虫为雌雄异体，通过两性交配后，精子与卵子结合，雌性产下受精卵，每粒卵发育成一个子代个体的生殖方式。

2.孤雌生殖 （单性生殖/处女生殖）：卵不受精也能正常发育成新个体

昆虫的变态

d.复变态



幼虫（若虫）期

幼虫和若虫两个称呼的区别：一般来说，若虫会经历几个龄期才达到成熟，外形和成虫比较接近；而幼虫则会通过蛹化转变为成虫，通常出现在蛹化之前，外形和成虫很不相同。

足类型：

  1. 无足型
  2. 多足型
  3. 寡足型
    蛹类型：
  4. 被蛹
  5. 离蛹
  6. 围蛹

成虫期

羽化：成虫从其前一虫态脱皮而出的过程。

补充营养：对成虫的性成熟不可或缺的营养。

性二型现象：同种昆虫雌雄成虫除生殖器官不同外，在外部特征（触角、体色、身体大小）有明显不同的现象。

多型现象：同种昆虫在同一性别的个体中出现不同类型的分化现象。

休眠与滞育

休眠：由不利环境条件引起的暂时性生长发育停滞现象，当不利环境条件消除时，能立刻恢复生长发育。

滞育：昆虫在光周期和温度变化等环境因子的诱导下，通过体内基因控制的发育停滞现象，称之为滞育。

昆虫的习性

食性

植食性、肉食性、腐食性、杂食性。
单食性昆虫——只取食一个种的植物
寡食性昆虫——只取食一个科的植物
多食性昆虫——取食不同科的植物

假死性

趋性

昆虫对外界刺激所产生的趋向或背向行为活动，其中趋向活动又称正趋性，背向活动称为负趋性

保护色和拟态

群集、扩散、迁飞

龄期

两次蜕皮之间所经历的时间称为龄期
刚孵化为1龄，每蜕皮一次则+1龄（x=1 x += 1）
高龄幼虫往往伴随着暴食期，其食量占一生的90%以上，故而防治虫害时应在低龄时进行防治。
