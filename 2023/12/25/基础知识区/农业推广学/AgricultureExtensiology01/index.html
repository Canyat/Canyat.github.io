行为的产生

行为的概念

人的行为指的是，在一定社会环境中，在人的意识支配下，按照一定的规范进行并取得一定结果的客观活动

构成要素

  • 主体：是人，农业推广中指的是农民
  • 客体：作用的对象（可以是人，也可以是物）
  • 世界观：即行为产生的状态，是在人的意识支配下的活动，而非无意识的活动。具有目的性、方向性和预见性

行为的特征

  1. 目的性：人的活动一般有预定的目的、计划和期望 # 最大，最基本的特征
  2. 调控性：人的行为受主观影响，即人的思维支配、调节和控制自己的行为
  3. 差异性：个体受外界影响产生不同的差异
  4. 可塑性：人的行为受教育（社会、家庭、学校的教育/毒打）产生变化
  5. 创造性：人的行为是积极地认识和改造世界的创造性活动

行为理论及其应用

需要层次理论

需要是引起动机进而导致行为产生的根本原因

# 马斯洛把人类的需要分为五个层次（如下图）

  1. 生理需要：维持生命种族延续所必须
  2. 安全需要：心理和物质上的安全保障
  3. 社交需要：感情和归属
  4. 尊重需要：自尊和受别人尊重
  5. 自我实现需要：人类最高层次的需要

动机理论

动机：由内在需要及外来刺激引发的，是一个人为满足某种需要而进行的活动的意念或想法

作用

  1. 始发作用：动机是产生行为的动力[前提]
  2. 导向作用：动机是行为的指南针[保证行动与目标的一致性]
  3. 强化作用：增强或减弱行为速度的催化剂

激励理论及其应用

行为激励即激发人的动机，使人产生内在的冲动，朝向期望的目标前进的心理活动过程（调动人的积极性）

期望理论

激励力量（M） ＝ 效价（V）× 期望值（E）

  • 期望值：是指一个人对某个目标能够实现的可能性（概率）的估计

  • 效价：指目标价值，即目标有无作用

    # 太高的目标作用低，容易达到的目标作用高

# 激励力量越强，激励力量就越不弱（是废话，但不是废话）

应用

  1. 正确确定推广目标，科学设置推广项目
  2. 认真分析农民心理，热情引导农民兴趣
  3. 提高推广人员素质****，积极创造良好推广环境**
