前言

看到这么一张图，属实有点没绷住

之前总是听人说：如果一个男孩子连资源都找不到，也许该怀疑一下他的能力

当然这肯定是玩笑话，搜索能力除了因人而异，还和生活的环境息息相关

假如你让一个从来没接触过电脑的人自学c语言，这个难度肯定是比有计算机基础的人学习c要大得多

故而，我也不藏私，把我这个超过十五年网龄的老网民到现在还在用的搜索工具放出来啦~

虽然在食用指南说了，但是这里再补充一下，载不了图建议科学上网或是加速器加速github

观前提醒：分类都只是按个人的感觉分的如果和你的想法不一样那你一定是对的没有必要和我理论什么东西相信自己就好

文字阅读创作类

文学类

一、古诗文网

顾名思义就是找古诗文的，如果遇到很冷门的诗句，没准可以在这里碰碰运气，大部分都能找到

二、Z-Library

这个网站非常多电子书，说是世界上最大的电子图书馆也不为过，但是容易寄，之前已经被搞了，建议直接用app

x rpc_token = ‘Y6YOKzhKasoo’  video_url = f’{url}’  # 下载地址output_name = f’{name}.mp4’  # 下载文件名s = client.ServerProxy(‘http://localhost:16800/rpc')s.aria2.addUri(‘token:’ + rpc_token, [video_url], dict(out=output_name))​Python

需要….过多解释吗？

四、好读网

认识这个网站也是机缘巧合，里面有非常多文学名著，绝大部分为繁中书籍，可以竖向阅读，如果想读金庸古龙等老武侠书，不妨去看看

五、鸠摩搜书

这个太经典了，不多评价，总之就是棒

六、书籍知识库

好用，但是用得不多

七、其他网址导航（用的比较少，但是个别书籍有奇效）

网页导航其一：可以看zlib镜像站

熊猫搜书：直接用这个的话可以找到一堆资源，相比上面的话需要自己甄别，好用是挺好用，但是运气不好的话也很费时间，对于搜索小白而言不是非常推荐，所以我放得比较后面来说

网文轻小说

# 虽说我也想把他归类到文学类里面，毕竟网络文学也是文学嘛，但是这个东西又很特别，受众非常广不说，质量也参差不齐，有些文字实在是很难跟“文学”二字沾边，所以就单独拿出来了

一、番茄小说网

看爽文，我推荐番茄！配合跳广告的话，无论是听书还是阅读，都没啥阻碍，而且小说也是免费的，有着不错的讨论环境。但是想靠阅读赚钱，建议别整，这东西看个乐子得了

二、哔哩轻小说

我的邻家天使、拖油瓶就是在这里看的，app体验一般（作为一个有强迫症的人，我的使用要求还是比较严格的），属于是勉强流畅使用的那种，资源也多，可用！

三、魔笔小说

热门资源非常好找，冷门的话还是要碰碰运气，想看冷门的建议看我下面这个

四、轻小说文库

这是个重量级，为什么我要放在这么后面而不是第一个说呢，我列出了几点：

  1. 不支持epub/mobi下载，不能边看文本边看插图
  2. 页面比较古早，如果不是老网民可能会不适应
  3. 不能在线阅读
  4. 看完上面三条我觉得不用再加了

但是但是但是！这是重点，这个网站能活到现在，可以说巨大的库存功不可没，即便是非常冷门的小说，只要有汉化，基本上都能找到（如果你找不到，那估计就是太过冷门以至于几乎所有汉化组都看不上XD）

五、次元狗

中规中矩，可以试试

六、有度中文网

不好评价，但是我也在这里找到过书

七、其他搜索方式

其实这个地方应该挺多的才对，但是当我今天整理的时候，已经寄了好多个网站了，毕竟这种资源都是涉及版权问题的，若不是打工人，谁不想支持正版呢？以后如果还能找到的话，我会往这里加，保持更新（大概）

创作使用类

# 提笔写作的时候用的，或是参考，或是协助，以前还有个押韵助手，但是今天看的时候寄了。写作的时候不懂的东西大部分都是度娘解决，所以我也提供不了太多

一、深言达意

如果你也提笔忘字、欲言无词，想要说些什么高深的话或是引经据典表达一些观点，但是无奈文学积累和站主一样贫瘠，可以试试这个黑科技，用一次爱一次

当然我之前用的是他的前身据意查句，虽然不再维护，但是还能用，需要也可以直接点进去

二、搜韵

这个搜诗词相关的格律，典故都很好用，甚至可以在线检查格律（朋友说的，喜欢诗词的可以试试）

三、汉典

当字典用

学术类

# 我是学生，免费给我，作为一个大学牲，写论文太常见了，但凡事都使用gpt，很容易变成一个废人，所以姑且放上一些，对文献检索、论文写作有用的网站吧

文献检索类

一、公益学术平台

咱们国家自己建立的，反知网垄断的平台，自然要放第一个

二、两个图书馆

  • 广东省立中山图书馆

  • 四川省图书馆

具体怎么用，看我前面的文章《如何使用图书馆知网资源》（在“工具区-科研工具”的tag下，请自行寻找）

三、中国知网

学校一般有买、有教，就不用多说了

四、SCI-HUB中文社区

这个用法我更建议你直接b站，有点难讲清楚

五、谷歌学术

好东西，但不是特别好，国内难上，酌情使用

六、X-Mol学术平台

也是个好东西，建议自己操作试试

七、NCBI

搞生物的，大名鼎鼎了，应该和PubMed是同个东西

八、百度学术

特有的杂乱风格ui，但是有些文献这里找比较方便，具体的自己试试

文献阅读写作类

# 不用动脑子乱写 提高效率用的，毕竟作为学术垃圾生产工具的大学牲而言，对水课论文多懂点脑子都是对自己高考的不尊重

一、通义智文

可以理解为，可国内访问的copilot

二、通义千问

我对阿里的产品好感度还是不错的

三、文心一言

虽然有人喷，但是人家毕竟也是个ai产品，新版本对于自然语言的处理已经是相当好了，不吹不黑，我虽然很少用，但是每次用的结果都还是满意的
